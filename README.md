# 🌍 PolyGlot | Modern Translator

PolyGlot is a sleek, modern, and user-friendly translator application designed to provide seamless English translation into **Finnish, Bangla, and Persian**. It features a premium "Glassmorphism" interface and supports both text and voice interactions.

![App Screenshot](https://github.com/rahmanmdmustafizur958-ai/PolyGlot-Translator/raw/main/screenshot.png) *(Note: Add your screenshot here once uploaded)*

## ✨ Features

- **🚀 Instant Translation**: Translate English text into Finnish, Bangla, or Persian with a single click.
- **🎙️ Voice Input (STT)**: Speak in English and watch it convert to text automatically using the Web Speech API.
- **🔊 Voice Output (TTS)**: Listen to the translated text in the native accent of the target language.
- **📱 Responsive Design**: Fully optimized for both desktop and mobile devices.
- **🎨 Premium UI**: Beautifully crafted with soft colors, rounded buttons, and smooth micro-animations.
- **📋 One-Click Copy**: Easily copy your translated result to the clipboard.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS, JavaScript (Standalone Version)
- **Framework**: React + Vite (Original Source)
- **API**: MyMemory Translation API (Free & Unlimited)
- **Voice**: Web Speech API (SpeechRecognition & SpeechSynthesis)

## 🚀 How to Run Locally

Since this version is built as a standalone static application, you can run it without any complex setup!

### Method 1: Using `npx serve` (Recommended)
1. Open your terminal in the project folder.
2. Run:
   ```bash
   npx serve
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Method 2: Direct Open
Simply double-click the `index.html` file in the root directory to open it in your favorite web browser.
